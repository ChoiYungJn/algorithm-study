## 커밋 규칙
### Format

#### [Keyword]: {출처} {난이도} {제목}
**ex) [solve] 백준 silver 4 숫자 카드2**

### Keyword
- `solve`: 풀이에 성공한 새 문제 추가
- `fail`: 풀이에 실패한 새 문제 추가
- `improve`: 새로운 성공 풀이를 추가한 문제
- `docs`: readme 등 문서 수정
